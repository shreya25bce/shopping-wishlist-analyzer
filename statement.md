# Project Statement

**Shopping Wishlist Cost Analyzer**

## Problem Statement

students often need a simple tool to maintain and analyse their personal shopping wishlists, track budgets, and understand spending patterns. the tools that already exist are either too complex or online-only or lack multi user support usually. this project provides an offline, lightweight python tkinter application to solve this problem

## Scope of the Project

* enable users to maintain wishlists with item details stored in csv format
* allow category analysis of spending using charts like piechart
* provide budget tracking features with color coded alerts
* supports multiple users and also an admin mode for viewing
* offers basic item operations such as add edit/delete/search/sort & import/export.
* ensures the interface is simple, responsive, and user friendly.

## Target Users

* students managing personal expenses and wishlists
* user needing an offline desktop tool for small scale budget tracking
